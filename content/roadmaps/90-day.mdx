## How to use this roadmap

This is the highest-coverage track in the platform. It is designed for candidates who need enough time to re-sequence fundamentals, deepen weak areas, and still finish with repeated mock practice.

Use the roadmap as a working schedule:

- complete the phase goal before moving on
- keep one implementation block and one speaking-practice block every week
- bookmark the topics that still feel slow or fuzzy after first review

## Exit criteria

By the final phase, you should be able to answer common ML theory prompts without notes, explain RAG and evaluation trade-offs clearly, and design one ML system and one GenAI system end to end.
