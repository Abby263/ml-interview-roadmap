# Code of Conduct

This project is for learning and interview preparation. Keep collaboration
professional, constructive, and specific.

Expected behavior:

- be respectful when reviewing content or code;
- give actionable feedback with examples;
- assume good intent, but correct misinformation directly;
- avoid harassment, personal attacks, discrimination, or spam;
- do not post private candidate, company, or interview-confidential material.

Unacceptable behavior may result in issue or PR closure, comment deletion, or
maintainer blocking.

Report conduct concerns to the maintainers through the repository.
