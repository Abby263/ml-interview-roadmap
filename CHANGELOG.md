# Changelog

Notable changes to ML Interview Roadmap are tracked here.

This project follows a lightweight release process:

- pull requests describe user-facing changes;
- CI must pass before merge;
- tags matching `v*` create GitHub releases through `.github/workflows/release.yml`.
